## 내용정리
- 충성스러우면서 불변 -> 어떤 경우에도 항상 동일한 엔티티를 대표하면서 불변
- 객체는 실제 엔티티(real-life entity)의 대표자
	- 실제엔티티 - 객체의 가시성 범위 밖에 존재하는 모든것( 즉 실제 객체가 대표하고있는 것)
- 객체의 상태는 실제 객체의 좌표이다
- 모든 객체는 식별자, 상태, 행동을 포함한다.
- 불변객체의 식별자는 불변객체의 상태 그 자체이다
## 생각 정리
